How to Use

You need 21 to post on and interact with this site

If you have one you can you can use the site by downloading the linkshare-client.py here

Right now it costs 1000 satoshis to post a link and 500 to comment. It costs 100 satoshis to upvote or downvote a link and 50 to upvote or downvote a comment. All of the proceeds from upvotes are immediately paid to whoever posted the link or comment

You can find the server source code in this repository

Commands:

Add a link:

python3 linkshare-client.py add_link <url>

optional parameters:
--title = <title to link>
--image = <url to image to be displayed with the posts>

Example used to add the first link:
python3 linkshare-client.py add_link http://21.co --image="https://assets.21.co/landing/img/img_device.jpg" --title="The 21 Bitcoin Computer"

Upvote a link:

python3 linkshare-client.py upvote < index>

Downvote a link:

python3 linkshare-client.py downvote < index>

Comment on a link:

python3 linkshare-client.py comment <link index> <comment>

Upvote a comment:

python3 linkshare-client.py upvote_comment <link index> <comment index>

Downvote a comment:

python3 linkshare-client.py downvote_comment <link index> <comment index>
